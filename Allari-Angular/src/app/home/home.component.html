That is a web application using Angular for the front-end and .NET Core for the back-end. The application still fulfills all the specified requirements:

Main Page Layout

  • The main page of the application features a website layout with a:
    • Header
    • Navigation bar
    • Content sections on the right and left
    • Main content area
    • Footer

Header Section

In the header section, there is a digital clock displaying the current time, with real-time updates for seconds, minutes, and hours. This clock is implemented using Angular.

Navigation Bar

This section prominently displays the full name of the developer. It also includes a project description. Additionally, there is a drop-down button that, when clicked, lists the elements requested from Api. This listing is achieved through an asynchronous call using Angular, communicating with the .NET Core back-end for data retrieval.

Content on the Right

The content on the right side of the page features an image slider with previous and forward buttons, allowing users to navigate through a collection of images.

Content on the Left

On the left side of the page, there are indicators that display the X and Y mouse positions on the screen, with Angular providing real-time feedback to the user.

Footer

Finally, the footer of the page includes a link to the Allari webpage.
